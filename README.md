# Teslo API

1.Clonar Projeto
2. ``` npm i ```
3. Clonar o arquivo ```.env.template``` e renomear ```.env```
4. Trocar as variaveis de retorno
5. Levantar a base de dados
```
docker-compose up -d
```

6. Levantar ```npm run start:dev```
